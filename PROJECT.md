\# Ledger



\## Vision



Ledger is a premium Android expense tracker focused on speed, beauty and automation.



The goal is to make recording expenses almost invisible to the user.



\---



\# Principles



\- Offline First

\- Privacy First

\- Zero Ads

\- Fast

\- Beautiful

\- Minimal

\- Smooth

\- Secure



\---



\# Core Features



\- SMS Parsing

\- Notification Parsing

\- Merchant Learning

\- Smart Categories

\- Budgets

\- Analytics

\- Powerful Search

\- Backup \& Restore



\---



\# Architecture



MVVM



Clean Architecture



Repository Pattern



Single Activity



Jetpack Compose



Room Database



\---



\# Development Rules



\- No duplicated code.

\- Every feature must have tests where practical.

\- UI must remain responsive.

\- Never use Double for currency.

\- Money is stored in minor units.

\- Every transaction has its own currency.

\- Business logic never lives inside Composables.
