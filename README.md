\# Ledger



A premium offline-first Android expense tracker built with Kotlin and Jetpack Compose.



\## Goals



\- Beautiful UI

\- Automatic expense tracking

\- SMS parsing

\- Notification parsing

\- Smart categorization

\- Offline-first

\- No ads

\- No cloud dependency



\## Tech Stack



\- Kotlin

\- Jetpack Compose

\- Material 3

\- Room

\- Hilt

\- Coroutines

\- Flow

\- DataStore

\- Navigation Compose

\- Vico Charts
