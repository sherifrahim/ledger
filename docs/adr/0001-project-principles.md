# ADR-0001 — Ledger Project Principles

Status: Accepted

## Purpose

Ledger is a local-first Android personal finance application focused on speed, clarity, and reliability.

## Principles

1. Offline First
   Every core feature must work without internet.

2. Local Ownership
   User data belongs to the user.

3. Transactions are immutable records.
   Corrections create new records rather than silently changing history where practical.

4. Every screen has one visual focus.

5. Components before screens.

6. Buildable after every sprint.

7. Main branch is always releasable.

8. Currency is stored per transaction.

9. Money is stored as Long minor units.
   Never Double.

10. UI should prioritize information density without feeling crowded.
